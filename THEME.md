## Ant Design 组件主色

Ant Design 未配置自定义 ConfigProvider token，当前使用其默认主色 `#1677ff`。
项目实际主色为 `#2563eb`，两者视觉接近但存在轻微差异，如需统一可在 `main.ts` 中配置 ConfigProvider：

```ts
// main.ts 示例
import { ConfigProvider } from 'ant-design-vue'

app.use(ConfigProvider, {
  theme: {
    token: {
      colorPrimary: '#2563eb',
    },
  },
})
```

## 配置文件位置

| 文件 | 说明 |
|------|------|
| `src/style.css` | 全局基础样式、滚动条颜色、页面背景色 |
| `src/components/AppSidebar.vue` | 侧边栏完整配色（背景、激活、角色徽章） |
| `src/main.ts` | Ant Design 组件库引入（当前无自定义 token） |

> 当前项目颜色全部为硬编码，暂无集中的 CSS 变量或 token 文件。
