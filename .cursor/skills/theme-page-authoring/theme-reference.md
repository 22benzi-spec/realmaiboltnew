# Theme Reference

This file summarizes the current project theme so page-authoring tasks can apply it quickly.

If this file and the repo-root `THEME.md` ever differ, `THEME.md` wins.

## Core Palette

- Page background: `#f5f7fa`
- Primary text: `#1a1a2e`
- Sidebar background: `#1a1a2e`
- Primary / button / link / active: `#2563eb`
- Sidebar active text: `#60a5fa`
- Sidebar active background: `rgba(37, 99, 235, 0.25)`
- Logo gradient: `linear-gradient(135deg, #2563eb, #3b82f6)`

## Status Colors

- Success: `#059669`
- Sales badge text: `#34d399`
- Warning: `#d97706`
- Finance badge text: `#fbbf24`
- Danger: `#dc2626`
- Admin badge text: `#f87171`
- Secondary action: `#7c3aed`

## Borders And Support

- Border: `#e5e7eb`
- Light border: `#f0f0f0`
- Scrollbar track: `#f0f2f5`
- Scrollbar thumb: `#c1c7d0`
- Scrollbar thumb hover: `#a0a9b8`
- Secondary text: `#6b7280`
- Placeholder / disabled text: `#9ca3af`

## Role Badge Mapping

- `业务员`: background `rgba(5, 150, 105, 0.25)`, text `#34d399`
- `商务`: background `rgba(37, 99, 235, 0.25)`, text `#60a5fa`
- `财务`: background `rgba(217, 119, 6, 0.25)`, text `#fbbf24`
- `管理员`: background `rgba(220, 38, 38, 0.25)`, text `#f87171`

## Ant Design Vue

- Preferred `colorPrimary`: `#2563eb`
- If touching global visual tokens, keep them aligned with this value unless the user requests a theme refresh.
