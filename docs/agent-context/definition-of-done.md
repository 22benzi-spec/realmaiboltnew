# 完成定义

任务只有在满足下面条件时，才应被视为完成。

## 必须满足

- 改动范围与用户目标一致
- 已说明风险等级
- 已运行要求的本地验证
- 结果没有明显违反 `AGENTS.md` 与 `docs/agent-context/`
- 若涉及高风险变更，已明确需要人工确认

## 对代码任务的默认要求

- 能编译或通过项目规定的构建检查
- 新增规则、脚本、CI 时有对应文档说明
- 文案与对人说明优先中文
- 稳定字段与脚本接口保持英文键名

## 不算完成的情况

- 只描述了方案，没有实际落地
- 没有运行验证
- 页面能点开，但核心逻辑未完成
- 高风险改动没有明确审批提示
- reviewer 已指出明显问题但未处理
